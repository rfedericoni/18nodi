# 18 Nodi Senigallia

Sito statico promozionale per lo stabilimento balneare 18 Nodi di Senigallia.

## File principali

- `index.html`: struttura, SEO, dati strutturati e contenuti.
- `styles.css`: layout responsive e visual design.
- `script.js`: menu mobile e stato header allo scroll.
- `assets/`: logo e immagini locali usate dal sito.

## Fonti usate

- Spiagge.it: descrizione, servizi e indirizzo.
- P2 Design Studio: concept progettuale e fotografie dello stabilimento.
- Restaurant Guru e ReteImprese: telefono, orari, indirizzo e riferimenti social.
- Facebook e Instagram ufficiali indicati: linkati nel sito; i contenuti non erano leggibili via fetch non autenticato.

Prima della pubblicazione definitiva conviene verificare con il titolare orari, disponibilità stagionale e autorizzazione d'uso delle foto.
